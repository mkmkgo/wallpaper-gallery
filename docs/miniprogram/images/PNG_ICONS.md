## 使用方法

1. 点击上面的链接下载对应的PNG图标
2. 将下载的图标保存到 `images` 目录中
3. 确保文件名称与 `app.json` 中配置的一致

## 图标规格

- 尺寸：81×81 像素
- 格式：PNG
- 风格：简约线性图标
- 颜色：
  - 未选中状态：灰色 (#666666)
  - 选中状态：紫色 (#667eea)

## 备用方法

如果上面的链接无法使用，您可以使用以下在线工具转换：

1. [SVG to PNG Converter](https://svgtopng.com/)
2. [Convertio](https://convertio.co/svg-png/)
3. [Online-Convert](https://www.online-convert.com/)

步骤：
1. 上传 `images` 目录中的 SVG 文件
2. 设置尺寸为 81×81 像素
3. 下载转换后的 PNG 文件
4. 按照上面的命名规则重命名文件
