## US-001: 浏览首页滚动展示

**作为** 微信用户
**我希望** 在首页浏览水平滚动展示的壁纸
**以便** 我可以快速发现精选壁纸

### 验收标准（EARS格式）

#### AC-1: 水平展示

**模式**: SHALL
```
系统应在首页显示带有5-6张壁纸缩略图的水平滚动展示
```

**Given-When-Then** (用于BDD测试):
- **Given**: 我打开超赞壁纸小程序
- **When**: 首页加载
- **Then**: 我看到带有5-6张壁纸缩略图的水平滚动展示

---

#### AC-2: 查看更多按钮
**模式**: WHEN

**Given-When-Then** (用于BDD测试):

- **Then**: 我被带到精选壁纸页面
